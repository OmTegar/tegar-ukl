Om Tegar- SETUP DEBIAN

Pastikan anda sudah berhasil melakukan instalasi debian 11, dan sudah masuk pada menu setup debian pada bagian host name sebagai contoh kita gunakan nama - nama kelas - no.absen. lalu click continue.

Code

TegarDitoXT5-33

Domain Name

setelah mengisi host name kalian akan di mintai untuk mengisi domain

Code

moklet.TegarDitoXT5-33.sch.id

password ROOT user

setelah kalian menentukan nama domain kalian, langkah berikutnya kalian akan di minta untuk mengisi password root sebanyak 2 kali, NOTE wajib di ingat ingat untuk password root tersebut karena akan sangat sering di gunakan di step berikutnya.

Code

rootT533

password LOGIN user

setelah kalian menentukan password root , kalian juga akan di minta untuk mengisi password login / password user sebanyak 2 kali, NOTE wajib di ingat ingat untuk password login tersebut karena itu adalah password untuk memasuki debian kalian.

Code
